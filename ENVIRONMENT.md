# Environment specification

This document describes the software environment used to develop and run the
PsychoPy experiment for the auditory semantic 1-back pilot study.

The goal is to ensure reproducibility and facilitate deployment on the MRI
scanner computer.

---

## Operating system

- OS: Microsoft Windows 11 Home
- Version: 10.0.26100.26100

---

## PsychoPy

- PsychoPy version: **2023.2.3**
- Installation type: **Standalone**
- Launch mode: Builder / Runner GUI

Experimental scripts are developed directly in Builder (`.psyexp`) and run using the PsychoPy standalone Python environment.

---

## Python (used by PsychoPy)

- Python version: **3.8.10**
- Python executable: **C:\Program Files\PsychoPy\python.exe**

Experimental output files (`data/`) are excluded from version control.
